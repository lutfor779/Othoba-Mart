# Othoba Mart

## Run Locally

Clone the project

```bash
  git clone https://github.com/johirulislamnishat/othoba-mart-frontend.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
